
Bley Vinhos - Site estático gerado
---------------------------------
Arquivos neste pacote prontos para publicar no GitHub Pages.

Instruções rápidas:
1. Crie um repositório no GitHub.
2. Faça upload de todos os arquivos deste ZIP para a branch 'main'.
3. Nas configurações (Settings) do repositório -> Pages, selecione branch 'main' e folder '/' e salve.
4. Aguarde alguns minutos e seu site estará ativo em https://<seu-usuario>.github.io/<repo>

Para editar produtos:
- Abra products.json e edite/adicione produtos (campos: id, name, desc, category, price, image).
- Substitua imagens na pasta /images (referencie-as em products.json como 'images/nome.jpg').
